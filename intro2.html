iPhone 3G

아이폰 3G는 아이폰의 후속 모델이다. 아이폰 3G는 아이폰의 3번째 모델이라는 뜻이 아니라, 1세대 아이폰에 3G 모바일 인터넷 기술을 접목시켰다 하여 붙여진 이름이다.
아이폰 3GS는 아이폰 3G의 성능 개선 모델이며, 아이폰 4는 아이폰 시리즈 중 4번째 모델이란 뜻이고 4G와는 연관이 없다.
제조사 애플(Apple)
시리즈 아이폰(iPhone)
형태 바 형태(Bar)
발매일 미국 : 2008년 7월 11일
한국 : 2009년 11월 28일
운영체제 출시 : 아이폰 OS 2.0
최신 : iOS 4.2.1
전원 일체형 리튬 이온 배터리 1,150 mAh
통신 방식 4중 대역 GSM (850/900/1800/1900 MHz) GPRS, EDGE[1] 3중 대역 UMTS, HSDPA 3.6 Mbps (850/1900/2100 MHz)[2] ,
와이파이 IEEE 802.11 b/g, 블루투스 2.0+EDR
연결방식 USB 2.0, 3.5 mm 이어폰 잭
CPU 412 MHz로 언더클럭 된 620 MHz 삼성 32비트 RISC ARM 1176JZ(F)-S v1.0 [3][4]
GPU PowerVR MBX Lite 3D GPU[5]
화면 3.5" TFT-LCD 320 × 480 (HVGA) (163 ppi, 18 비트(262,144색))
입력 멀티터치 터치스크린 화면, 헤드셋, 근접 조도 감지 센서, 3방향 가속도계[6] 전자 나침반[7]
RAM 128 MB eDRAM[8]
내장 메모리 8 GB, 16 GB 플래시 메모리
후면카메라 200만 화소, 위치정보 기능
크기 & 무게 세로: 115.5mm, 가로: 62.1mm, 두께: 12.3mm, 133 g
후속기종 아이폰 3GS

iPhone 3GS

아이폰 3GS는 아이폰 3G의 후속제품이다.
ARM 11 코어기반 CPU에서 ARM Cortex-A8 코어기반 CPU로 바뀌는 등 여러가지 하드웨어의 성능이 향상되었다. (디스플레이와 같은 부가적인 요소는 많이 바뀌지 않았다.)
제조사 애플(Apple)
시리즈 아이폰(iPhone)
형태 바 형태(Bar)
발매일 미국 : 2009년 6월 9일
한국 : 2009년 11월 28일
운영체제 출시 :아이폰 OS 3.0
최신 : iOS 6.0.1
전원 일체형 리튬 이온 배터리 1,219 mAh[1]
통신 방식 Wi-Fi IEEE 802.11 b/g, 블루투스 2.1, 4중 대역 GSM (850/900/1800/1900 MHz), GPRS, EDGE[2], HSDPA 7.2 Mbps
연결방식 USB 2.0, 3.5 mm 이어폰 잭
CPU 600 MHz로 언더클럭 된 833 MHz 삼성 S5PC100 ARM Cortex-A8[3][4]
GPU PowerVR SGX535

입력 멀티터치 터치스크린 화면, 헤드셋, 근접 조도 감지 센서, 3방향 가속도계[5], 전자 나침반[6]
RAM 256 MB eDRAM[3]
내장 메모리 8 GB, 16 GB, 32 GB 플래시 메모리
후면카메라 300만 화소, 동영상, 위치정보, 자동 초점, 화이트 밸런스, 노출 기능
크기 & 무게 세로: 115.5mm, 가로: 62.1mm, 두께: 12.3mm, 135g
